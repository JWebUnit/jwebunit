How do I create a TestCase for my web application?

jWebUnit uses two approaches creating test cases, inheritence and delegation. The simplest way is to inherit from WebTestCase rather than junit.framework.TestCase. 
import net.sourceforge.jwebunit.WebTestCase;

public class ExampleWebTestCase extends WebTestCase {
    private static final String BASE_URL = "http://localhost:8080/yourwebapp";
    private static final String RESOURCE_BUNDLE_NAME = null;

    public ExampleWebTestCase(String name) {
        super(name, BASE_URL, RESOURCE_BUNDLE_NAME);
    }
}
The alternative is to include an instance of the WebTester class in you TestCase and delegate navigation and assertions to it. This is provided for flexibility. 
import junit.framework.TestCase;
import net.sourceforge.jwebunit.WebTester;

public class ExampleWebTestCase extends TestCase {
    private static final String BASE_URL = "http://localhost:8080/yourwebapp";
    private static final String RESOURCE_BUNDLE_NAME = null;
    private WebTester tester;

    public ExampleWebTestCase(String name) {
        super(name);
        tester = new WebTester(BASE_URL, RESOURCE_BUNDLE_NAME);
    }
}
