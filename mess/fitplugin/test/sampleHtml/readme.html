This is the sample site that jwebfit tests run.

The WebFixtureTest will use DirectoryRunner to locate all .fit files in the test/testInput folder. That will invoke Fit on all the tables using the WebFixture in source, and produce the results as html in test/testOutput. If any of the fit tests fail, one (1) test failure will be reported by JUnit.

The sample site with servlets can be started using net.sourceforge.jwebunit.fit.RunSite (compiled with test classes).

The test setUp looks for this file named readme.html to assert that the test site is OK.
